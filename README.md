# Sales Dashboard - Power BI

## Overview
Built an interactive Power BI dashboard to analyze retail sales performance.

## Key Features
- KPI tracking (Sales, Profit, Quantity)
- Sales trend analysis
- Top products and regional performance
- Category-based profit insights
- Interactive slicers for filtering

## Tools Used
- Power BI
- DAX
- Excel (data cleaning)

## Key Insights
- Sales grew consistently over 4 years
- Central region generated highest revenue
- Technology category contributed most profit
